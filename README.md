# My_Anime_Repository

This app will allow users to search and display anime information. Users will also have the option to add an anime to a watch list. The top 10 anime list will also display on the home page every time a user visits. 

## Tech Stack

- JavaScript
- Next.js
- React

## Data Handling

- Redux
- Local Storage
- Jikan API "https://docs.api.jikan.moe/"

## Getting Started

1. clone the repository to your own system.
2. Open the repository with your IDE.
3. run "npm install"
4. run "npm run dev"
5. Open up localhost on your browser.

## There is no API key necessary to use the API.

## Search Bar

1. enter an anime name using letters or numbers only.
2. A card displaying some anime data will populate the page.

## Top 10 Anime Grid

1. The grid will automatically render on the home page.
2. As long as the API is updated, the top 10 anime grid may be different the next time you visit the page.

## Navbar

1. On the top left corder of the navbar, you can route to that specific page.

## Watch list

1. If you clicked on the green add button on the bottom right of any anime card, you will see that anime listed on the watch list page.
2. Feel free to click the trash bin button to delete the anime from the watch list.
