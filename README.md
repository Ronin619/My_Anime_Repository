Getting Started
clone the repository to your own system.
Open the repository with your IDE.
run "npm install"
run "npm run dev"
Open up localhost on your browser.
There is no API key necessary to use the API.
Search Bar
enter an anime name using letters or numbers only.
A card displaying some anime data will populate the page.
Top 10 Anime Grid
The grid will automatically render on the home page.
As long as the API is updated, the top 10 anime grid may be different the next time you visit the page.
Navbar
On the top left corder of the navbar, you can route to that specific page.
Watch list
If you clicked on the green add button on the bottom right of any anime card, you will see that anime listed on the watch list page.
Feel free to click the trash bin button to delete the anime from the watch list.
